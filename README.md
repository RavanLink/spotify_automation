# 🎧 Spotify Automation System (Python)

A Python-based automation system for managing and enhancing Spotify workflows. This project leverages the Spotify Web API to automate playlist management, track interactions, and user-defined actions efficiently.

---

## 🚀 Features

- 🔄 Automated playlist creation & updates  
- ❤️ Auto-save (like) tracks  
- 🎵 Fetch and process user listening data  
- ⏱️ Scheduled automation tasks  
- ⚙️ Custom rule-based automation  
- 🔐 Secure OAuth authentication with Spotify  

---

## 🐍 Tech Stack

- **Language:** Python 3.x  
- **API:** Spotify Web API  
- **Auth:** OAuth 2.0  
- **Libraries:**
  - requests  
  - spotipy (optional)  
  - python-dotenv  
  - schedule / cron  

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/ravanlink/spotify_automation.git

# Navigate into the project
cd spotify-automation

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate


```
